# Gas sation offers

## Creator
Christos Chartomatsidis, 2023 
## Synopsis
This web-app is my answer to the final assignment in the course of web development of my last year in the Hellenic Open University (Computer Science). It uses PHP, html, js, css, and SQL for the database, as well as xml for data schema validation. 

## Description
The user can enter the app as a regular guest, as a gas station owner or as admin.
- As a guest, the user can check out what special price offers each gas-station has.
- As a gas station owner, the user logs in, and can set, alter or remove any offers, set an end-date, and download the relevant xml file.
- As admin, the user can add/remove registered gas stations.
